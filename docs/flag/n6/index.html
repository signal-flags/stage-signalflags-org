Variants

Click on an image below to download it as n6.svg , or download all flag images

Default pennants have a 6:4 aspect ratio and are 150% of the width of default alphabet flags.

Pennants in the 'square' set are the same as default pennants with a 6:4 aspect ratio. They are are twice the width of square flags.
